# Venba Calc

Free financial calculators by [Venba Apps](https://apps.venba.dev).

**Live:** https://calc.venba.dev

## Calculators

| Calculator | Description | Link |
|---|---|---|
| TN Pension Calculator | Tamil Nadu Govt Pension Calculator (TAPS 2026) — Compare CPS lump sum vs monthly pension. Bilingual Tamil/English. | [Open](https://calc.venba.dev/tn-pension/) |

## Tech Stack

- Plain HTML + Tailwind CSS (CDN)
- Hosted on GitHub Pages with custom domain (`calc.venba.dev`)

## Adding a New Calculator

1. Create a new folder (e.g., `emi/`)
2. Add `index.html` inside it
3. Update the landing page (`index.html`) to link to it
4. Push to `main` — GitHub Pages auto-deploys
